# Seleção (`if...else`)

```js
if (idade > 18) {
  console.log("pode dirigir");
}
```

## if braceless

- brace: {}
- less: menos

Se tiver apenas um comando a ser executado dentro do `if`, nao eh necessario o uso de chaves.

```js
if (idade > 18) console.log("pode dirigir");
else console.log("nao pode dirigir");
```
