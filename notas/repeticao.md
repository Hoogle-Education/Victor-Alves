# Repeticao

- inicializacao/ponto de partida
- verificacao/limite
- atualizacao

```js
let count = 1; // inicializacao

while (count <= 5) {
  // verificacao
  console.log(count);
  count++; // atualizacao
}
```

## for

```js
for (;;) /*inicializacao*/ /*verificacao*/ /*atualizacao*/ {
  // faca algo
}
```

```js
for (let i = 1; i <= 5; i++) {}
```
